# Learning Hub

A centralized learning platform built with **Java**, **Spring Boot**, and **Thymeleaf** where students can access course materials, submit assignments, attend video sessions, and receive feedback.

---

## Features

- **User Management** – Students and instructors with role-based access.
- **Course Materials** – Upload and view documents, PDFs, and other learning resources.
- **Assignment Submission** – Submit tasks and track deadlines.
- **Video Sessions** – Join live or recorded video sessions.
- **Feedback System** – Receive and provide feedback on assignments and sessions.
- **Dashboard** – Overview of courses, tasks, and feedback.

---

## Technology Stack

- **Backend:** Java, Spring Boot  
- **Frontend:** Thymeleaf, HTML, CSS  
- **Database:**  MySQL  
- **Build Tool:** Maven  
- **Version Control:** GitHub

# 🎓 Learning Hub

A centralized learning platform built with **Java**, **Spring Boot**, and **Thymeleaf** where students can access course materials, submit assignments, attend video sessions, and receive feedback efficiently.



## ✨ Features

- 🧑‍🎓 **User Authentication & Roles** – Students and Instructors with role-based access  
- 📚 **Course Material Management** – Upload and view PDFs, documents, and learning resources  
- 📝 **Assignment Submission** – Submit tasks, track deadlines, and view feedback  
- 🎥 **Video Sessions** – Join live or recorded sessions  
- 💬 **Feedback System** – Instructors provide feedback on submissions  
- 📊 **Dashboard** – Personalized view of courses, tasks, and feedback  



## 🛠 Technology Stack

- **Backend:** Java , Spring Boot 
- **Frontend:** Thymeleaf, HTML5, CSS3  
- **Database:**  MySQL  
- **Build Tool:** Maven  
- **Version Control:** Git  



## ⚡ Setup & Installation

### Prerequisites

- Java 17 or above  
- Maven 3.8+  
- MySQL/PostgreSQL (optional for production)
